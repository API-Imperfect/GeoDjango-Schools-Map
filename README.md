## Run the Application

```
cd GeoDjango-Schools-Map
run the command: make build
navigate to localhost:8080
```

If you prefer not using Make files

```
cd GeoDjango-Schools-Map
run the command: docker-compose --build -d --remove-orphans
navigate to localhost:8080
```
